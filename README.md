# EasyMEP

EasyMEP is a free productivity add-in for Autodesk Revit MEP workflows.

## Supported Revit versions

Autodesk Revit 2021 through 2027.

## Download

Open the [Releases](https://github.com/AmrCodesHere/EasyMEP-Downloads/releases) page and download the latest `EasyMEP-*.msi` installer.

## Installation

1. Close all running Revit instances.
2. Download and run the MSI installer.
3. Start Revit and open the EasyMEP ribbon tab.

## File verification

Every release publishes the installer's SHA-256 checksum in its release notes.
